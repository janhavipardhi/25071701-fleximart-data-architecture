# FlexiMart Data Architecture Project

## Student ID
25071701

## Overview
End-to-end data engineering project for FlexiMart including:
- ETL pipeline
- Relational database design
- Business SQL queries
- MongoDB NoSQL analysis
- Data warehouse star schema

## Folder Structure
- etl/ → ETL scripts & data quality report
- sql/ → Database schema & business queries
- nosql/ → MongoDB analysis & queries
- warehouse/ → Data warehouse design
- data/ → Raw datasets

## Tech Stack
- Python (Pandas, MySQL Connector)
- MySQL
- MongoDB
- SQL

## How to Run
1. Place raw CSV files in `/data`
2. Run `etl/etl_pipeline.py`
3. Execute SQL scripts in MySQL
4. Run MongoDB queries using Mongo shell
